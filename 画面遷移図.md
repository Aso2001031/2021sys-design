```uml
@startuml

[*] -down-> トップページ


トップページ -right-> ログイン :「Sing in」をクリック
トップページ -down-> 会議参加
トップページ -down-> 会議作成


ログイン : entry/メールアドレス,パスワード入力
ログイン : do/ログイン認証
ログイン -left-> トップページ :「Start」をクリック
ログイン -right-> 会員登録 :「アカウントを持っていない方はこちら」をクリック

会員登録 : entry/メールアドレス,パスワード入力
会員登録 : do/登録処理
会員登録 -up-> 登録完了 : 「Regist」をクリック

登録完了 -> トップページ :「Start」をクリック

@enduml
```
